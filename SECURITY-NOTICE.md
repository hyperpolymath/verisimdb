# Security Standards

This site follows hyperpolymath security standards:
- Post-quantum cryptography (Kyber-1024, Dilithium5, SPHINCS+)
- IPv6-only (IPv4 terminated)
- TLS 1.3 only (HTTP/1.1 terminated)
- HTTP/3 + QUIC
- WCAG 2.3 AA/AAA compliance
- Scott Helme A+ security headers

See: https://github.com/hyperpolymath/polystack/blob/main/poly-ssg/sites-build/SECURITY-STANDARDS.scm
