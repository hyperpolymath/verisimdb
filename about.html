About VeriSimDB

Hexad-modality polyglot database with 6 unified stores

Overview

VeriSimDB is an open-source project developed by the hyperpolymath community. We are committed to building high-quality, ethical, and accessible software that respects user privacy and promotes open knowledge.

Mission

Our mission is to create tools and technologies that empower developers and users while maintaining the highest standards of:

  • Accessibility (WCAG 2.3 AA/AAA compliance)
  • Security (best practices, responsible disclosure)
  • Privacy (data sovereignty, minimal collection)
  • Ethics (human rights, anti-slavery, anti-money laundering)
  • Open Source (PMPL-1.0-or-later license)

Technology Stack

This project dogfoods hyperpolymath's technology standards:

  • Website: poly-ssg (static site generator)
  • Hosting: Cloudflare Pages (edge delivery)
  • Security: HTTP 430 + consent-aware-http protocol
  • License: PMPL-1.0-or-later (Palimpsest License)
  • Development: ReScript, Rust, Idris2, Elixir, Gleam
  • Verification: Formal methods, dependent types, property testing

Community

  • Discussions - community forum
  • Contact - feedback and inquiries
